# Home
## Ranking system

| Score | Description |
|-------|-------------|
| 1     | I have never heard of this skill or domain. |
| 2     | I have heard about this skill, but I don’t know how to use it or put it into practice. |
| 3     | I understand the basic concepts of this skill. |
| 4     | I can apply the basic concepts of this skill. |
| 5     | I can use this skill independently for simple and routine tasks. |
| 6     | I can apply this skill effectively for complex tasks and I am able to solve common problems without help. |
| 7     | I use this skill effectively in various complex situations and I can optimize processes or working methods. |
| 8     | I master this skill and I am able to train and mentor others in this field. |
| 9     | I am recognized as an internal expert, capable of developing new methods or improving existing practices thanks to this skill. |
| 10    | I am a recognized leader in this field, able to innovate and propose unique, cutting-edge solutions that set new standards. |
